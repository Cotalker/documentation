Por defecto, existen tres _tipos de usuarios_ o _cargos_:
- [**Realizador de Tareas**](/docs/products/forms/configuration/users/user_types#executor): Es el encargado de ejecutar tareas y envía formularios con data correspondiente.
- [**Coordinador de Tareas**](/docs/products/forms/configuration/users/user_types#coordinator): Agenda tareas y delega responsables.
- [**Administrador de Tareas**](/docs/products/forms/configuration/users/user_types#admin): Crea formularios, agrega ubicaciones técnicas e ingresa nuevos usuarios.
